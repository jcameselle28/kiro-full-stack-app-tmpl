Visual Style Guide

Light, airy SaaS design language · Orange + Red primary, Green / Purple / Gray / White secondary · Plus Jakarta Sans

Design System Reference · v1

About This Guide

This is a project-agnostic design system reference. It documents the visual language only — color tokens, typography, components, shape, and depth — with no product-specific content. To create a style guide for a new project, start from these tokens and adjust the palette, type, and component samples to fit that project's brand.
